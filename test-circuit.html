Circuit.js JQuery plugin to perform LCD display.



Download on Github

Easy to use

We start with a canvas:


	<body><canvas></canvas></body>

After including JQuery and circuit.js, we can perform LCD display.

      
	<script>
		$(document).ready(function() {
			$('canvas').circuit('123456789', {
				foreground: 'red',
				background: 'yellow',
				gridcolor: 'green',
				motion: 'yes',    
				interval: 1000
			});
		});
	</script>
